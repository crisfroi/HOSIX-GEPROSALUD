PASO 1: Arrancar Docker ICD
  Terminal → Copia esto:
  
  docker-compose up -d
  
  O si no tienes docker-compose:
  
  docker start icd11-server

PASO 2: Verificar que Docker está corriendo
  Abre en navegador: http://localhost:8090/swagger/index.html
  
  ✅ Si ves "Swagger UI" → Docker OK
  ❌ Si ves "Connection refused" → No está corriendo aún
